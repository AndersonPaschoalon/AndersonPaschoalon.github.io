SIMITAR (SniffIng, ModellIng, and TrAffic geneRation)


SIMITAR (SnIffing, ModellIng and TrAffic geneRation) is a tool for creating synthetic and realist network traffic.

It uses actual network traffic (live captures or pcap files) for creating a flow-oriented traffic model. This model is used for creating network traffic, and is provided as a parameter for the traffic generator. It is self-configured because you don't have to care about adjusting stochastic parameters such as exponential rates, protocol, ports, or traffic rates as input parameters. It is all contained inside the traffic model created.

The traffic generator is not attached to a specific technology. It has a flow scheduler, which creates a thread for each flow. As the traffic generator engine we can use applications or low-level C/C++ libraries. Currently we have extended it to Libtins (a C++ packet crafter library) and Iperf (network benchmark tool).
